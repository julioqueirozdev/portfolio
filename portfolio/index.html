Projetos

Uma seleção dos trabalhos dos quais ja participei.

Projeto Android

O projeto Android é uma página estática, feita inteiramente com HTML e CSS que conta a história da criação do mascote do Android. O projeto foi desenvolvido no curso de HTML e CSS do Curso em Vídeo.

Projeto Cordel

O projeto Cordel é uma página estática criada com HTML e CSS que exercita conceitos de configuração de imagens de fundo e apresenta de forma simples e fluida o poema Cordel Moderno de Milton Duarte. O projeto foi desenvolvido como desafio no curso de HTML e CSS do Curso em Vídeo.

Calculadora de IMC

A calculadora de IMC é um projeto desenvolvido utilizando HTML e CSS e arquitetura MVC para organizar a estrutura de Javascript que permite que a aplicação armazene dados dos usuários localmente no navegador, calcule e exiba seus dados e IMC em tela. É um projeto focado em entender e aplicar Javascript Orientado a Objetos idelaizado pelo Professor Edson Maia.

Projeto Login

O projeto Login é uma página de login responsiva, criada com HTML e CSS que foca em aplicar conceitos de criação de formulários e métodos de layout flexbox. O projeto foi feito como desafio no curso de HTML e CSS do Curso em Vídeo.

Jogo Detona Ralph

O jogo Detona Ralph é baseado no filme de mesmo nome e foi criado utilizando HTML, CSS e Javascript. Nele, precisamos acertar o Ralph nas janelas com cliques do mouse dentro do limite de tempo e sem perder as vidas disponíveis. É um projeto desenvolvido no Bootcamp para Devs Frontend da DIO.

Jogo da Memória

O jogo da Memória, criado com HTML, CSS e Javascript, simula o funcionamento de um jogo da memória da vida real onde precisamos combinar pares de emoji idênticos. É um projeto desenvolvido no Bootcamp para Devs Frontend da DIO.

Pokedex

A Pokedex é um projeto criado em HTML, CSS e Javascript que exibe pequenos cards com informações dos Pokémons da primeira geração do joguinho. Neste projeto, são exercitados conceitos de consumo de API REST. É um projeto desenvolvido no Bootcamp para Devs Frontend da DIO.

Jogo Yu-Gi-Oh!

O jogo Yu-Gi-Oh é um jogo no estilo jo-ken-po, criado com HTML, CSS e Javascript. Nele, o jogador precisa escolher uma carta em sua mão, que tem valores de pedra, papel e teroura, enquanto o computador faz o mesmo do seu lado e o vencedor é decidido e a potuação computada. É um projeto desenvolvido no Bootcamp para Devs Frontend da DIO.

Piano Virtual

O simulador de piano virtual, criado com HTML, CSS e Javascript, funciona com uma versão simples de um piano que pode ser tocado tanto com o mouse quanto com as keys to teclado correspondentes às teclas do piano. É um projeto desenvolvido no Bootcamp para Devs Frontend da DIO.
